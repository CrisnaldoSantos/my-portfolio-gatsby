# my-portfolio-gatsby
Meu site de portfólio desenvolvido com GatsbyJS: https://www.crisnaldocarvalho.com.br/

### Descrição:
<p>O GatsbyJS é um framework capaz de gerar sites estáticos atilizando componentes React e GraphQL. Com o objetivo de prover velocidade e SEO, no momento de construção do site consultas GraphQl são realizadas e renderizadas nos componentes, tornando possível construir sites estáticos através de dinamismo.</p>
<p>
Aumenta consideravelmente a velocidade de produção do projeto, bem como, adaptar e modificar o conteúdo, uma fez que as informações estão dispostas no site metadata do arquivo config e apenas a alteração do mesmo se faz necessária para atualizar o conteúdo.</p>
